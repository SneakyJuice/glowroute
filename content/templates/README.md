# GlowRoute Content Templates

This directory contains reusable content templates for programmatic SEO content generation on GlowRoute. Each template includes placeholders for dynamic variables, ensuring consistency and scalability across the platform.

## Available Templates

1. **city-landing.md** - "[City] Med Spas" landing pages
2. **treatment-page.md** - "[Treatment] in [City]" pages (e.g., Botox, fillers, semaglutide)
3. **comparison.md** - "Best Med Spas in [City]" comparison articles
4. **faq.md** - "[Treatment] FAQ" or "Med Spa FAQ: [City]" pages
5. **listicle.md** - "Top 10 Med Spas in [City]" articles

## Variable Reference

Each template uses the following placeholders:

- `{{CITY}}`: The city name (e.g., "Tampa")
- `{{STATE}}`: The state abbreviation (e.g., "FL")
- `{{TREATMENT}}`: The specific treatment (e.g., "Botox")
- `{{CLINIC_COUNT}}`: The number of clinics in the city (e.g., "25")
- `{{YEAR}}`: The current year (e.g., "2026")
- `{{TOP_CLINIC_1}}`, `{{TOP_CLINIC_2}}`, `{{TOP_CLINIC_3}}`: Top-rated clinics in the city
- `{{AVG_RATING}}`: The average rating of clinics (e.g., "4.8")

## Usage Guide

1. **Select a Template**: Choose the appropriate template for your content type.
2. **Fill in Variables**: Replace placeholders with actual data (e.g., `{{CITY}}` → "Tampa").
3. **Review Content**: Ensure the content flows naturally and meets SEO guidelines.
4. **Publish**: Upload the filled template to your CMS or platform.

## Quality Guidelines

- Maintain a natural, conversational tone.
- Avoid AI-generated "gook"—write for humans.
- Ensure all facts are accurate and up-to-date.
- Include a clear call-to-action (CTA) linking to the GlowRoute directory.
